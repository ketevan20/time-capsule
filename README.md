# Time Capsule 

**Time Capsule** is a modern interactive web experience built with **Next.js**, **Tailwind CSS**, and **Framer Motion** — designed to take users on a journey through humanity’s greatest achievements. From ancient milestones to modern breakthroughs, explore history in a visually engaging way.

 Live Demo: https://time-capsule-ten-lovat.vercel.app/  
 Repository: https://github.com/ketevan20/time-capsule

## Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/ketevan20/time-capsule.git
cd time-capsule
```

### 2. Install dependencies
 ```bash
npm install
```
### 3. Run the development server
 ```bash
npm run dev
```
Open http://localhost:3000 
 to view it in the browser
